# BuiManhHieu_22003155_CK1

Project Expo/React Native với Expo Router và NativeWind (Tailwind CSS).

## Cấu trúc Project

```
src/
├── app/
│   ├── _layout.tsx      # Root layout
│   ├── index.tsx        # Entry point
│   └── (tabs)/
│       ├── _layout.tsx  # Tabs layout
│       └── home.tsx     # Home screen
├── components/          # Components (trống)
├── db/                  # Database (trống)
├── hooks/               # Custom hooks (trống)
├── theme/               # Theme/Colors (trống)
├── types/               # TypeScript types (trống)
└── global.css           # Global styles
```

## Cài đặt

```bash
npm install
```

## Chạy ứng dụng

```bash
npm start
```

Sau đó chọn:
- `a` - Android emulator
- `i` - iOS simulator
- `w` - Web browser
- Quét QR code với Expo Go app

## Dependencies đã cài

- Expo Router
- NativeWind (Tailwind CSS)
- React Native Paper
- Expo SQLite
- TypeScript

## Bắt đầu

Project đã được reset về trạng thái rỗng. Bạn có thể bắt đầu xây dựng ứng dụng mới từ đây.
